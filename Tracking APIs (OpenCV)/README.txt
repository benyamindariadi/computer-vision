This simple code contains various object detection algorithms in opencv.
The object detection API are:
1.TrackerBoosting
2.TrackerMIL
3.TrackerKCF
4.TrackerTLD
5.TrackerMedianFlow

Instructions: 
1.Enter the API 
2.Select the object do you want to be tracked (with mouse), select and drag until the object covered by the blue rectangle. 
3.Press enter 
4.The object tracker is working 
5.Press esc to close the window
